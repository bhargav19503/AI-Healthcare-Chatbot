# AI Healthcare Chatbot

This project is an AI-based healthcare chatbot that predicts diseases based on user symptoms using machine learning.

## Features
- Disease prediction using Random Forest
- Symptom analysis
- Chatbot interaction
- Provides precautions

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas, NumPy

## How to Run
1. Install requirements
2. Run: python app.py
3. Open: http://127.0.0.1:5000/
